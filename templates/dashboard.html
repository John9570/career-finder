Career Finder Overview

How To Use

With great power comes great responsibility. Career Finder's data can currently be manipulated by anybody. Currently Career Finder uses the challenge data (the raw file can be uploaded). Reuploading this isn't necessary. The search form uses the CSV file to populate the searchable majors and goes through the file for each search which is why you might notice a small delay on each search.

Magic Behind The Math

Currently Career Finder uses the following algorithm: A major contains data relating to people in various occupations within one column of data. This data can be further split into seperate categories. Since there is only one category though, one dependent variable, all one needs to do is sum the amount of data (people) in a particular major and divide that summation by the number in each category in each occupation. When a particular sample is select (for example "All") this only further refines the one category.

For example, say the highest data points within Accounting is within the occupation Accounting. Career Finder would say you have the highest probability of going into Accounting after graduating with your degree in accounting because most people do that. It will also list the relevant facts, what percent of people go into what occupation and how much data it has to work with to determine that percentage.
